Fluid & adaptive columns

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md, lg & xlg viewport widths.

{% highlight html %}
{% endhighlight %} {% highlight less %} @import "grid/grid.less"; @import "grid/_variables.less"; .container:extend(.row) { .column-name:extend(.col-xs-6, .col-sm-4, .col-md-3, .col-lg-2, .col-xlg-1) {} .column-other:extend(.col-xs-6, .col-sm-4, .col-md-6, .col-lg-8, .col-xlg-10) {} } {% endhighlight %}
