Offsetting columns

Move columns to the right using .col-md-offset-* classes. These classes increase the left margin of a column by * columns.

{% highlight html %}
{% endhighlight %} {% highlight less %} .container:extend(.row) { .column-name:extend(.col-xs-2) {} .column-other:extend(.col-xs-2, .col-xs-offset-8) {} } {% endhighlight %}
